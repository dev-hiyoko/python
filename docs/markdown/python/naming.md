# 命名法

## 変数と関数とモジュールとパッケージ

スネークケース

```python
my_variable = 1

def my_function():
    pass

my_module.py
import my_module
```

## プライベート変数と関数

アンダースコアで始まる

```python
_my_variable = 1

def _my_function():
    pass
```

## 定数

大文字とアンダースコア

```python
MY_CONSTANT = 1
```

## クラス

キャメルケース

```python
class MyClass:
    pass
```
